рҹ“– SPICE нҠңнҶ лҰ¬м–ј

SPICE кё°ліё мҶҢмһҗ(V/I/R/L/C/M)мҷҖ мӢңл®¬л Ҳмқҙм…ҳ мў…лҘҳ(.dc / .tran / .ac / .noise) мӮ¬мҡ©лІ•мқ„ мқөнһҲлҠ” 5к°Җм§Җ нҡҢлЎң. нҒҙлҰӯн•ҳл©ҙ editorм—җ лЎңл“ңлҗ©лӢҲлӢӨ.

SPICE мӢӨмҠө нҺҳмқҙм§Җ мӮ¬мҡ©лІ•

н•ҷмғқмқҙ мһҗмң лЎӯкІҢ SPICE netlistлҘј мһ‘м„ұВ·мӢңл®¬л Ҳмқҙм…ҳн•ҳлҠ” мӢӨмҠө лҸ„кө¬мһ…лӢҲлӢӨ. мӢңл®¬л Ҳмқҙм…ҳмқҖ лёҢлқјмҡ°м Җ м•Ҳмқҳ ngspice WASMмңјлЎң мӢӨн–үлҗҳл©°, м„ңлІ„лЎң м „мҶЎлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

1. кё°ліё мӮ¬мҡ©

  • мҷјмӘҪ editorм—җ netlistлҘј мһ‘м„ұн•©лӢҲлӢӨ. лӘЁлҚёмқҖ мһҗлҸҷмңјлЎң /model.libм—җ лЎңл“ңлҗҳлӢҲ .lib "/model.lib" MOSлЎң м°ёмЎ°н•©лӢҲлӢӨ.
  • м§Җмӣҗ н•ҙм„қ: .tran, .dc, .ac, .noise. н•ң netlistм—җ м—¬лҹ¬ к°ң мһ‘м„ұ к°ҖлҠҘ.
  • мҳӨлҘёмӘҪ нғӯ(Tran/DC/AC/Noise)мқ„ м„ нғқн•ҳкі  в–¶ Runмқ„ лҲ„лҘҙл©ҙ н•ҙлӢ№ н•ҙм„қл§Ң ngspiceлЎң мӢӨн–үлҗ©лӢҲлӢӨ.

2. нғӯлі„ лҸ…лҰҪ мӢӨн–ү

к°Ғ Run лІ„нҠјмқҖ ngspiceлҘј лі„лҸ„ м„ём…ҳмңјлЎң нҳём¶ңн•©лӢҲлӢӨ:

  • нҳ„мһ¬ нғӯмқҳ н•ҙм„қ lineл§Ң лӮЁкё°кі  лӢӨлҘё .tran/.dc/.ac/.noiseлҠ” мқјмӢң м ңкұ°
  • н•ҷмғқмқҳ .control...endc лё”лЎқлҸ„ мқјмӢң м ңкұ° (мһҗлҸҷ wrapperлЎң лҢҖмІҙ)
  • мһҗлҸҷ wrapperлЎң кІ°кіјлҘј /<analysis>.txtм—җ dump

мҳҲ: Tran нғӯмқҳ Run лІ„нҠј лҸҷмһ‘:

* (.dc / .ac / .noise / .control м ңкұ°)
.tran <н•ҷмғқмқҙ м“ҙ мӢңк°„ нҢҢлқјлҜён„°>
.control

3. н•ҙм„қлі„ н‘ңмӢң

  • Tran: мӢңк°„(s) vs V/I (linear)
  • DC: sweep ліҖмҲҳ vs V/I (linear)
  • AC: мЈјнҢҢмҲҳ(log) vs |H| dB & вҲ H deg (Bode plot 2лӢЁ)
  • Noise: мЈјнҢҢмҲҳ(log) vs noise spectral density (log-log) вҖ” 1/f noise лҜёнҸ¬н•Ё

4. ngspice лЎңк·ё

ngspiceмқҳ stdout/stderr м „мІҙк°Җ мўҢмёЎ н•ҳлӢЁ рҹ“ң ngspice лЎңк·ём—җ н‘ңмӢңлҗ©лӢҲлӢӨ. л””лІ„к№… мӢң м°ёкі н•ҳм„ёмҡ”.

5. нӮӨліҙл“ң лӢЁм¶•нӮӨ

  • F5 лҳҗлҠ” Ctrl+Enter: нҳ„мһ¬ нғӯмқҳ н•ҙм„қ мӢӨн–ү
  • Esc: мӢӨн–ү мӨ‘м§Җ
  • Alt+1: Tran нғӯ м „нҷҳ + мӢӨн–ү
  • Alt+2: DC нғӯ м „нҷҳ + мӢӨн–ү
  • Alt+3: AC нғӯ м „нҷҳ + мӢӨн–ү
  • Alt+4: Noise нғӯ м „нҷҳ + мӢӨн–ү

лӢЁм¶•нӮӨлҠ” editor м•Ҳ/л°– м–ҙл””м„ңлӮҳ мһ‘лҸҷн•©лӢҲлӢӨ. лӢӨлҘё мһ…л ҘлһҖм—җ нҸ¬м»ӨмҠӨк°Җ мһҲмқ„ л•ҢлҠ” л№„нҷңм„ұнҷ”.

6. к·ёлһҳн”„ л§Ҳмҡ°мҠӨ мЎ°мһ‘

  • мҷјмӘҪ л“ңлһҳк·ё: к·ёлһҳн”„ panning (мқҙлҸҷ)
  • мҳӨлҘёмӘҪ л“ңлһҳк·ё: мҳҒм—ӯ м„ нғқ zoom (л“ңлһҳк·ён•ң мӮ¬к°Ғнҳ•мқҙ мғҲлЎңмҡҙ н‘ңмӢң мҳҒм—ӯ)
  • л§Ҳмҡ°мҠӨ нң : м»Өм„ң мң„м№ҳ мӨ‘мӢ¬мңјлЎң zoom in/out
  • лҚ”лё” нҒҙлҰӯ: zoom мҙҲкё°нҷ” (м „мІҙ лҚ°мқҙн„° мҳҒм—ӯмңјлЎң ліөк·Җ)

7. мҳҲм ң

н—ӨлҚ”мқҳ рҹ“– к°Өлҹ¬лҰ¬ лІ„нҠјм—җм„ң 5к°Җм§Җ SPICE нҠңнҶ лҰ¬м–јмқ„ лЎңл“ңн•  мҲҳ мһҲмҠөлӢҲлӢӨ. кё°ліё мҶҢмһҗ (V/I/R/L/C/M)мҷҖ мӢңл®¬л Ҳмқҙм…ҳ мў…лҘҳ (.dc / .tran / .ac / .noise) мӮ¬мҡ©лІ•мқ„ лӢӨлЈ№лӢҲлӢӨ.

8. мһҗлҸҷ м ҖмһҘ / лӮҙліҙлӮҙкё° / л¶Ҳлҹ¬мҳӨкё°

  • мһҗлҸҷ м ҖмһҘ: netlist В· нҢҢлқјлҜён„° мһ…л Ҙк°’ В· нҷңм„ұ нғӯ В· лЎңк·ё лҶ’мқҙк°Җ лёҢлқјмҡ°м Җ localStorageм—җ мһҗлҸҷ м ҖмһҘлҗ©лӢҲлӢӨ. лӢӨмқҢм—җ нҺҳмқҙм§ҖлҘј м—ҙл©ҙ мқҙм „ мһ‘м—…мқҙ к·ёлҢҖлЎң ліөмӣҗлҗ©лӢҲлӢӨ.
  • рҹ’ҫ лӮҙліҙлӮҙкё°: нҳ„мһ¬ м„Өм •мқ„ JSON нҢҢмқјлЎң лӢӨмҡҙлЎңл“ң. лӢӨлҘё мӮ¬лһҢкіј кіөмң н•ҳкұ°лӮҳ л°ұм—…мҡ©мңјлЎң мӮ¬мҡ©.
  • рҹ“Ӯ л¶Ҳлҹ¬мҳӨкё°: JSON нҢҢмқјмқ„ м„ нғқн•ҙ м„Өм •мқ„ ліөмӣҗ. нҳ„мһ¬ мһ‘м—…мқҙ лҚ®м–ҙм“°кё°лҗҳлӢҲ мЈјмқҳ.
  • мЈјмқҳ: мӢңл®¬л Ҳмқҙм…ҳ кІ°кіј(м°ЁнҠё лҚ°мқҙн„°)лҠ” м ҖмһҘлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ. ліөмӣҗ нӣ„ лӢӨмӢң в–¶ Run н•ҙм•ј м°ЁнҠёк°Җ лӮҳнғҖлӮ©лӢҲлӢӨ.

9. м ңн•ң

  • мӢӨн–үмқҖ 30мҙҲ timeout. AC sweepмқ„ л„Ҳл¬ҙ кҙ‘лҢҖм—ӯмңјлЎң мһЎмңјл©ҙ мӨ‘лӢЁлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • nested .dc sweepмқҖ 1м°Ё sweepл§Ң н‘ңмӢңлҗ©лӢҲлӢӨ.
  • н•ң лІҲм—җ н•ҳлӮҳмқҳ ngspice м„ём…ҳл§Ң мӢӨн–үлҗ©лӢҲлӢӨ (Stop лІ„нҠјмңјлЎң мӨ‘лӢЁ к°ҖлҠҘ).

10. SPICE мҶҢмһҗ / мҶҢмҠӨ syntax

ngspice manual кё°мӨҖ мҶҢмһҗ 1н–ү нҳ•мӢқ (мІ« кёҖмһҗк°Җ мҶҢмһҗ мў…лҘҳлҘј кІ°м •):

R<name> n+ n- <value>                      ; м Җн•ӯ       (мҳҲ: R1 in out 1k)
C<name> n+ n- <value> [IC=<v0>]             ; м»ӨнҢЁмӢңн„°    (мҳҲ: C1 out 0 100n)
L<name> n+ n- <value> [IC=<i0>]             ; мқёлҚ•н„°
D<name> n+ n- <model>                       ; лӢӨмқҙмҳӨл“ң
Q<name> nC nB nE [nS] <model>               ; BJT
M<name> nD nG nS nB <model> W=<w> L=<l>     ; MOSFET   (мҳҲ: M1 d g 0 0 nmos4 W=20u L=1u)

лҸ…лҰҪ/мў…мҶҚ мҶҢмҠӨ:

V<name> n+ n- <DC value> [AC <mag> [phase]] [stim]  ; м „м••мӣҗ
I<name> n+ n- <DC value> [AC <mag>] [stim]          ; м „лҘҳмӣҗ
E<name> n+ n- nc+ nc- <gain>                        ; VCVS (м„ нҳ•л§Ң; POLYлҠ” ngspice WASM лҜём§Җмӣҗ)
B<name> n+ n- V=<expr>                              ; behavioral source вҖ” л№„м„ нҳ•/мҲҳмӢқ мһ…л Ҙ
                                                     ; мҳҲ: B1 vcm 0 V = V(in1)+V(in2)/2

м „м••мӣҗ stim (PULSE / SIN / EXP / PWL / AC):

V1 in 0 DC 0 AC 1                                        ; AC л¶„м„қ/л…ёмқҙмҰҲмҡ©
V1 in 0 PULSE(V1 V2 TD TR TF PW PER)                     ; нҺ„мҠӨ (мҳҲ: PULSE(0 1 100u 1n 1n 0.5m 1m))
V1 in 0 SIN(VO VA FREQ TD THETA)                         ; м •нҳ„нҢҢ
V1 in 0 EXP(V1 V2 TD1 TAU1 TD2 TAU2)                     ; м§ҖмҲҳ мқ‘лӢө
V1 in 0 PWL(t1 v1 t2 v2 ...)                             ; piecewise linear

лӢЁмң„ prefix (лҢҖмҶҢл¬ёмһҗ л¬ҙкҙҖ, лӢӨл§Ң M = milli, Meg = mega):

T = 1e12   G = 1e9    Meg = 1e6   K = 1e3
M = 1e-3   U = 1e-6   N = 1e-9    P = 1e-12   F = 1e-15

мҳҲ: 1Meg = 1,000,000О©, 1M = 0.001О©. нқ”н•ң мӢӨмҲҳ.

лӘЁлҚё нҢҢмқј include:

.lib "/model.lib" MOS                ; ліё нҺҳмқҙм§Җм—җм„  ETRI PDKк°Җ /model.libлЎң мһҗлҸҷ л§ҲмҡҙнҠёлҗЁ
                                     ; MOS / DIODE / BJT / RES / CAP м„№м…ҳ м„ нғқ к°ҖлҠҘ
.include "/model.lib"                ; лқјмқҙлёҢлҹ¬лҰ¬к°Җ м•„лӢҢ мқјл°ҳ include

11. л¶„м„қ лӘ…л № syntax

* DC sweep вҖ” н•ң мҶҢмҠӨлҘј step лӢЁмң„лЎң sweep
.dc <src> <start> <stop> <step> [<src2> <s2> <e2> <st2>]
   мҳҲ: .dc VGS 0 3 0.005
       .dc VDS 0 5 0.05 VGS 0.5 1.5 0.25     ; nested (мҷёл¶Җ sweepл§Ң м°ЁнҠё н‘ңмӢң)

* Transient вҖ” мӢңк°„ мҳҒм—ӯ
.tran <tstep> <tstop> [<tstart> [<tmax>]] [UIC]
   мҳҲ: .tran 1u 3m
       .tran 10u 5m 1m            ; 1ms мқҙнӣ„л§Ң м¶ңл Ҙ
       UIC = мҙҲкё°мЎ°кұҙ(IC=) мӮ¬мҡ©, .op кұҙл„ҲлңҖ

* AC вҖ” мЈјнҢҢмҲҳ мҳҒм—ӯ (мҶҢнҳ•мӢ нҳё; .op нӣ„ small-signal linearization)
.ac <dec|oct|lin> <Npoints> <fstart> <fstop>
   мҳҲ: .ac dec 20 1 1Meg          ; decadeлӢ№ 20м җ, 1Hz~1MHz
       .ac lin 100 50 60          ; 50~60Hz 100м җ

* Noise вҖ” мһ…л Ҙ referred / м¶ңл Ҙ noise spectrum
.noise V(<output>[,<ref>]) <input_src> <dec|oct|lin> <N> <fstart> <fstop>
   мҳҲ: .noise V(out) Vin DEC 20 1 100Meg
       - input_srcм—җ л°ҳл“ңмӢң AC=1 лӘ…мӢң (.noiseлҠ” ACлҘј лӮҙл¶Җ мӮ¬мҡ©)
       - inoise_spectrum / onoise_spectrum л‘җ мӢ нҳё м¶ңл Ҙ

12. мһҗмЈј л§ҢлӮҳлҠ” мҳӨлҘҳ / нҢҒ

  • Noise input source has no AC value вҖ” .noise мһ…л Ҙ мҶҢмҠӨм—җ AC 1мқ„ м¶”к°Җ. мҳҲ: Vin in 0 DC 1.0 AC 1
  • MOSFET лӘЁлҚёмқ„ лӘ» м°ҫмқҢ вҖ” .lib "/model.lib" MOSк°Җ M мҶҢмһҗліҙлӢӨ мң„ мӨ„м—җ мһҲм–ҙм•ј н•Ё.
  • м Җн•ӯмқҙ кұ°мқҳ 0О©мңјлЎң лҸҷмһ‘ вҖ” 1M мҚјлҠ”м§Җ нҷ•мқё. megaлҠ” 1Meg.
  • л§Ҳм§Җл§ү мӨ„м—җ .end лҲ„лқҪ вҖ” мһ…л Ҙ лҒқмқ„ ngspiceк°Җ мқёмӢқ лӘ»н•ҙ лҸҷмһ‘м җл§Ң н’Җкі  л¶„м„қмқ„ кұҙл„ҲлңҖ.
  • POLY мӮ¬мҡ© мӢң лҸҷмһ‘ м•Ҳ н•Ё вҖ” ngspice WASMмқҖ XSPICE кё°л°ҳ POLYлҘј м§Җмӣҗн•ҳм§Җ м•ҠмқҢ. B sourceлЎң лҢҖмІҙ: 
    * м°ЁлҸҷ мҰқнҸӯкё° мһ…л Ҙ (vcm + vid/2, vcm вҲ’ vid/2)
Bip ip 0 V = V(vcm) + 0.5*V(vid)
Bin in 0 V = V(vcm) - 0.5*V(vid)
  • AC кІ°кіјк°Җ нҸүнҸүн•ң м§Ғм„  вҖ” AC 1мқҙ мһ…л Ҙ мҶҢмҠӨм—җ м—Ҷмңјл©ҙ нҡҢлЎңк°Җ small-signalлЎң л“ӨлңЁм§Җ м•ҠмқҢ.
  • Nested DC sweep вҖ” ngspiceлҠ” л‘җ ліҖмҲҳ лӘЁл‘җ sweepн•ҳм§Җл§Ң ліё нҺҳмқҙм§Җ м°ЁнҠёлҠ” 1м°Ё ліҖмҲҳл§Ң н‘ңмӢң.
  • мЈјм„қ вҖ” мІ« мӨ„мқҖ мһҗлҸҷмңјлЎң title (мЈјм„қмңјлЎң мІҳлҰ¬). ліёл¬ём—җм„ңлҠ” *лЎң мӢңмһ‘н•ҳлҠ” мӨ„, лҳҗлҠ” $В·; мқҙнӣ„к°Җ мЈјм„қ.
