SPICE 실습 페이지 사용법

    
학생이 자유롭게 SPICE netlist를 작성·시뮬레이션하는 실습 도구입니다.
       시뮬레이션은 브라우저 안의 ngspice WASM으로 실행되며, 서버로 전송되지 않습니다.


    
1. 기본 사용

    
    
      
  • 왼쪽 editor에 netlist를 작성합니다. 모델은 자동으로 /model.lib에 로드되니 .lib "/model.lib" MOS로 참조합니다.
    • 
      
  • 지원 해석: .tran, .dc, .ac, .noise. 한 netlist에 여러 개 가능.
    • 
      
  • ▶ Run 클릭 시 ngspice가 1회 실행되어 모든 해석을 순차 수행하고, 각 결과가 해당 탭에 표시됩니다.
    • 
    


    
2. 자동 wrapper

    
.control 블록을 직접 쓰지 않으면, 검출된 해석에 따라 다음이 자동 추가됩니다:


.control
run
setplot tran1   ; wrdata /tran.txt all   (.tran 있을 때)
setplot dc1     ; wrdata /dc.txt all     (.dc 있을 때)
setplot ac1     ; wrdata /ac.txt all     (.ac 있을 때)
setplot noise1  ; wrdata /noise.txt inoise_spectrum onoise_spectrum   (.noise 있을 때)
quit
.endc

    
직접 .control을 쓸 경우, 결과를 위 경로로 wrdata 해야 차트에 나타납니다.


    
3. 해석별 표시

    
    
      
  • Tran: 시간(s) vs V/I (linear)
    • 
      
  • DC: sweep 변수 vs V/I (linear)
    • 
      
  • AC: 주파수(log) vs |H| dB & ∠H deg (Bode plot 2단)
    • 
      
  • Noise: 주파수(log) vs noise spectral density (log-log) — 1/f noise 미포함
    • 
      
  • Log: ngspice stdout/stderr 전체
    •
